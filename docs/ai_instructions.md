# AI Usage Instructions (STRICT)

## 🎯 Purpose

Use AI as an assistant, not a replacement.

---

## 🧠 Core Rule

> I OWN THE LOGIC. AI HELPS WITH EXECUTION.

---

## ✅ Allowed AI Usage

* Convert Excel logic → Python
* Debug errors
* Suggest optimizations
* Generate boilerplate code
* Improve structure

---

## ❌ NOT Allowed

* Blind copy-paste code
* Generating full project without understanding
* Skipping debugging
* Letting AI decide architecture

---

## 🧩 Prompt Templates

### 1. Convert Excel Logic

"Convert this Excel logic into pandas code:
[PASTE LOGIC]"

---

### 2. Debug Error

"I get this error:
[ERROR]
Here is my code:
[CODE]
Explain the issue and fix it."

---

### 3. Optimize Memory

"This pandas code is using too much memory. Optimize it for low RAM environment."

---

### 4. Design Step

"I am building a pipeline: ingestion → processing → append → output.
Help me structure this step properly."

---

## ⚡ Workflow

1. Try yourself
2. Get stuck
3. Ask AI specific question
4. Understand response
5. Implement
6. Test

---

## 🚨 Golden Rule

If you cannot explain your code:
→ You did not build it
→ AI built it

That is NOT acceptable for this project.

---

## 🎯 End Goal

Be able to say:

"I built a data pipeline that processes real-world mainframe data and automates reporting."
