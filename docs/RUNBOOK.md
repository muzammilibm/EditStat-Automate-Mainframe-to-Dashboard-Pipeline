# Runbook

## ▶️ How to Run

```bash
python main.py
```

---

## 🔄 Daily Workflow

1. Place raw file in `/data/raw`
2. Run pipeline
3. Check `/outputs/reports`
4. Validate results

---

## ⚠️ Common Issues

### Memory Error

* Use smaller dataset
* Enable chunk processing

### Missing Columns

* Check input file format

---

## 🧪 Validation Checklist

* Data appended correctly
* No duplicates
* Excel generated properly
